# IGNITE Pre-Workout Drink Website

A landing page for IGNITE Pre-Workout Energy Drink.

## Features

- Responsive design for all screen sizes
- Animated scrolling banner
- Product showcase with floating animation
- Features grid highlighting product benefits
- Customer testimonials
- Contact form

## Structure

```
Pre-workout-Drink/
├── index.html          # Main website page
├── images/
│   └── ignite-can.png  # Product image
└── README.md
```

## Setup

1. Add the `ignite-can.png` product image to the `images/` folder
2. Open `index.html` in a web browser

- 200mg Clean Caffeine
